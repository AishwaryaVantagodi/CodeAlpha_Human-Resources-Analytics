📌 Project Title
HR Analytics & Real Estate Market Trends Dashboard (Power BI)
📄 Description
This project focuses on building interactive dashboards using Power BI to analyze and optimize business decisions in two domains: Human Resources and Real Estate.
The HR Analytics dashboard helps organizations monitor recruitment efficiency, employee performance, satisfaction, and attrition trends. It also uses predictive analytics to forecast hiring needs.
The Real Estate Market Trends dashboard provides insights into property pricing, demand patterns, and market dynamics to support investment and development decisions.
Both dashboards are designed to be interactive, data-driven, and easy to interpret for decision-makers.
